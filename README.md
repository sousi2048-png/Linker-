# Linker→

3D空間で展開される、ゲームルール改変ギミックと物体をつなげるパズルゲーム。

## 概要

**Linker→**は、ノードを接続してゲームのルールを動的に変更する3Dパズルゲームです。修飾語と名詞のノードを組み合わせることで、世界の法則を書き換え、障害を乗り越えていきます。

## キーワード

- **3D空間** - 立体的な環境でパズルを解く
- **ゲームルール改変ギミック** - ノードの接続で世界の法則が変わる
- **物体をつなげるパズル** - ノードを接続して新しい関係性を生み出す

## 操作方法

- **WASD** - キャラクター移動
- **マウス** - 視点移動（カメラ回転）
- **ESC** - メニューを開く/閉じる

## ゲームシステム

### ノードシステム

パズルは2種類のノードで構成されています：

- **修飾語ノード（黄色）** - 他のノードに属性や能力を付与する
- **名詞ノード（青色）** - ゲーム世界内の物体や概念を表す

### ノード接続の仕組み

ノード間の接続方向によって、ゲームのルールが変わります。

#### 例1: プレイヤーを操作可能にする

```
Player ← Operable
```

この接続により、`Player`（主人公）が操作可能になります。WASDキーで移動できます。

#### 例2: 障害物を操作可能にする

```
Player    Operable → Obstacles
```

この接続により、`Player`は操作できなくなりますが、代わりに`Obstacles`（障害物）を操作できるようになります。WASDキーで障害物を動かし、道を開けることができます。

### ノード解放システム

ゲームを進めるにつれて、新しいノードが解放されていきます。解放されたノードを使って、より複雑なパズルを解いていきます。

## 技術仕様

- **エンジン**: Unity 6000.2.12f1
- **プラットフォーム**: PC（Windows）

## プロジェクト構造

```
Assets/
├── Scripts/          # ゲームロジック
│   ├── RedBoxController.cs    # メインコントローラー
│   ├── DraggableNode.cs       # ノードのドラッグ処理
│   ├── NodeMergeManager.cs    # ノード結合管理
│   └── ...
├── Scenes/           # ゲームシーン
└── ...
```

## セットアップ

1. Unity 6000.2.12f1以上をインストール
2. このリポジトリをクローン
3. Unity Editorでプロジェクトを開く
4. `Assets/Scenes/SampleScene.unity`を開いて実行
